# SageInvest 제품 개요

## 프로젝트 정보

- **프로젝트명**: SageInvest
- **유형**: 웹 애플리케이션
- **주요 언어**: TypeScript
- **프레임워크**: Next.js
- **데이터베이스**: PostgreSQL
- **개발 키트**: MoAI-ADK

## 프로젝트 설명

SageInvest는 AI 기반 투자 어시스턴트 웹 애플리케이션입니다. TypeScript와 Next.js로 구축되며, PostgreSQL을 활용하여 사용자의 투자 데이터를 안전하게 관리합니다. AI 기술을 활용한 투자 분석과 자산 배분 추천을 통해 개인 투자자가 보다 현명한 투자 결정을 내릴 수 있도록 지원합니다.

## 타겟 사용자

SageInvest는 다음과 같은 사용자 그룹을 타겟으로 합니다:

- **개인 투자자**: 장기 투자를 목표로 하는 개인으로, 포트폴리오 관리와 자산 배분 최적화를 원하는 분들
- **액티브 트레이더**: 단기 매매를 통해 수익을 실현하는 트레이더로, 실시간 시세 데이터와 기술적 분석 도구를 필요로 하는 분들
- **일반 사용자**: 재무 관리에 관심이 있는 일인 사용자로, 쉬운 인터페이스와 직관적인 투자 가이드를 원하는 분들

## 핵심 기능

### 1. 포트폴리오 관리

- 계좌 및 보유 자산 통합 관리
- 실시간 포트폴리오 가치 추적
- 수익률 분석 (일일, 월간, 연간)
- 자산별 비중 및 리밸런싱 알림

### 2. 실시간 시세 데이터

- 주식, ETF, 암호화폐 실시간 시간
- 시장 지수 및 뉴스 피드
- 가격 알림 및 예약 설정
- 인터랙티브 차트 및 기술적 지표

### 3. 자산 배분 분석

- 현대 포트폴리오 이론 기반 자산 배분 추천
- 리스크 프로필별 맞춤 전략
- 시뮬레이션 및 백테스팅 도구
- 다각화 효과 분석

### 4. AI 투자 분석

- 머신러닝 기반 시장 예측
- AI 기반 투자 기회 발굴
- 리스크 평가 및 경고 시스템
- 투자 성과 보고서 자동 생성

## 개발 워크플로우

SageInvest는 MoAI-ADK의 SPEC 기반 TDD 워크플로우를 따릅니다:

1. **SPEC 작성** (`/moai:1-plan`): EARS 형식으로 요구사항 정의
2. **TDD 구현** (`/moai:2-run`): RED-GREEN-REFACTOR 사이클
3. **문서화** (`/moai:3-sync`): API 문서 및 아키텍처 문서 생성
